# Zora

### Zora.co



{% embed url="https://youtu.be/AylFJ3ie9qY" %}

1. Перейдите в свой профиль артиста на Zora.co и кликните на коллекцию, для которой вы хотите получить адрес контракта, в разделе Посты.

<figure><img src="../../.gitbook/assets/Screenshot 2024-10-17 at 14.12.49.png" alt=""><figcaption></figcaption></figure>

2. Кликните на любой из объектов в вашей коллекции, затем кликните на три точки под кнопкой 'Mint' и скопируйте адрес коллекции.

<figure><img src="../../.gitbook/assets/Screenshot 2024-10-17 at 14.15.41.png" alt=""><figcaption></figcaption></figure>

3. Вернитесь на CUR8, выберите Zora в поле Blockchain, установите поле Marketplace на Другое (Адрес контракта), вставьте адрес контракта в поле ID коллекции и заполните детали вашей коллекции для проведения верификации коллекции.

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-31 at 10.36.03.png" alt=""><figcaption></figcaption></figure>
