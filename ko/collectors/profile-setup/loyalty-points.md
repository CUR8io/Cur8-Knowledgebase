# 💲 로열티 포인트

CUR8 파워 사용자는 미션을 완료하여 리더보드에서 순위를 올리고 다양한 흥미로운 혜택과 보상에 접근할 수 있도록 로열티 포인트로 보상받습니다!

### 포인트는 어떻게 사용될까요?

\
미션을 완료하여 얻은 포인트는 다음과 같이 사용될 수 있습니다:\


* CUR8 생태계 내에서 더 큰 유틸리티를 가질 우리의 유틸리티 토큰 $CUR8
* 파트너 생태계 토큰
* CUR8의 프리미엄 구독 플랜에 대한 할인된 접근
